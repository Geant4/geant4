## Quick Reference

### Project Information

- **Name**: Geant4
- **Description**: C++ toolkit for simulating the passage of particles through matter
- **Version**: 11.4.0
- **License**: Geant4 Software License
- **Website**: https://geant4.cern.ch

### Repository Structure

```
geant4/
├── source/           # Main source code
├── examples/         # Example applications
├── cmake/            # CMake modules and configuration
├── config/           # Configuration files
├── environments/     # Environment setup scripts
├── packaging/        # Package creation scripts
├── ReleaseNotes/     # Release notes and changelogs
├── CMakeLists.txt    # Top-level build configuration
├── README.rst        # Project README
├── LICENSE           # License information
├── CONTRIBUTING.rst  # Contribution guidelines
└── CITATION.cff      # Citation information
```
